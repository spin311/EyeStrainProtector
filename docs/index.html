How to use the extension:

  1. Play audio for every rest reminder (chosen on 7)
  2. Show notification for every rest reminder
  3. Turn extension off / on
  4. After how many minutes do you want to take a rest
  5. Time for snooze button in minutes
  6. For how long should notification be shown
  7. Select audio file to be played
  8. Select volume for audio file
  9. Test audio and notification
  10. GitHub page for extension
  11. Donate with PayPal or any credit card

Make sure to pin the extension
